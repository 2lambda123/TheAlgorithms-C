Purpose:
    This program is for educational purpose.
    This is a terminal application.

How to Compile and run:
    All files can be compiled using the Makefile
    by simply typing make in the terminal.
    The make file will compile all the .c and .o
    file to make an executable called run.exe
    which will run the driver file, compare.c,
    to compare the differences between a Regular
    Binary Search Trees and AVL Trees.
